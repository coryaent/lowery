## Overview
[Lowery](https://caddiehalloffame.org/all-hall-of-fame-inductees/175-eddie-lowery) is [Caddy](https://caddyserver.com) compiled with plugins for [Docker](https://www.docker.com) and [Redis](https://redis.io).
